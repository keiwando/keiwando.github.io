Does Evolution run on Linux?

Linux is not an officially supported platform for Evolution, so there is no Linux build of Evolution ready to just download.

If you are interested in running Evolution on Linux, you will have to use Unity and the source code to create a Linux build yourself. I can't guarantee if or how much of Evolution will work on Linux. Certain features, such as file import and export are definitely not going to work.
